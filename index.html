AngularJs环境的准备

工具和库：

  • WebStorm
  • Chrome
  • AngularJs
  • Angular Material
  • bower
    为了使用bower，您可能还需要安装如下工具：
    cnpm - 国内用户推荐，淘宝的npm加速服务
    npm - Node.js的包管理软件
    Node.js - 基于Chrome V8引擎的本地/服务端JS运行环境

其他工具

在安装好Node.js后，可以通过Node.js的npm命令安装最新版本的npm/cnpm/bower(如果已经存在，自动安装最新版本)。
Windows
采用如下方式查看是否安装成功(或者更高版本)

$ npm -v
3.5.3

$ cnpm -v
3.4.0

$ bower -v
1.7.2

 npm install -g npm --registry=https://registry.npm.taobao.org
 npm install -g cnpm --registry=https://registry.npm.taobao.org
 npm install -g bower --registry=https://registry.npm.taobao.org
//可选，将npm默认设置从淘宝服务器上获取数据
npm config set registry "https://registry.npm.taobao.org"
