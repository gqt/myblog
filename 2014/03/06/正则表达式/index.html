正则表达式

Javascript
  • Javascript

什么是正则表达式

使用特定的符号所描述的一种规则

  • 举例：
    用户名只能输入数字、字母、下划线，同时数字不可以作为开头，同时用户名不能包含中文，同时长度必须在6-20之间

    var str = username;
    var reg=/^[a-zA-Z]\w+$/;
    if(reg.test(str)){

    //符合规则

    }
    最早起源于Perl语言

    为什么要使用正则表达式？

    可以大量的节省验证的工作量

如何创建正则表达式？

  • 第一种:
    var reg=new RegExp('goole','gi');
  • 第二种：
    /goole,gi/.test(str)

reg代表了一种规则
reg.test(‘xxxx’) 代表了使用该规则对字符串进行校验

字符串的match方法

var pattern = /good/ig;
var str='qingTeng is very cute!';
console.log(str.match(pattern));
console.log(str.match(pattern).length);

使用search来查找匹配数据

var pattern = /good/ig;
var str='qingTeng is very cute!';
console.log(str.search(pattern));

使用replace替换匹配到的数据

var pattern = /good/ig;
var str='qingTeng is very cute!';
console.log(str.replace(pattern,'hard'));

表示任意字符

var pattern = /good/ig;
var str='qingTeng is very cute!';
console.log(pattern.test(str));

匹配次数问题

  • *表示任意次数0或多次
  • ？表示匹配1次或0次
  • +表示至少1次
  • {m,n}表示至少m次
  • {m,}表示至少m次

字符匹配

[a-zA-Z0-9] 列举所有符号的情况，当情况过多时，使用-表示范围

开始和结束符

^
$
\d 数字
\D 非数字
\w 数字字母下划线
\W
\s 空白字符

检查邮政编码/共6位数字，第一位不能为0

/^[1-9]\d{5}$/

检查文件压缩包/xxx.zip\xxx.gz\xxx.rar\xxx.7z\xxx.tar

/^.+\.(zip|gz|rar|7z|tar)$/

删除多余空格

str.replace(/\s+/g,"")

删除收尾空格

str.replace(/^\s+/,"").replace(/\s+$/,"");
